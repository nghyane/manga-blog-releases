# manga-blog Releases

Public binary releases for manga-blog.

## Download Latest Version

```bash
# Clone this repo
git clone https://github.com/nghyane/manga-blog-releases.git
cd manga-blog-releases/latest

# Or download specific version
cd v1.0.0
```

## What's Included

- ✅ Compiled binary with embedded templates
- ✅ Static assets (CSS, JS, fonts, images)  
- ✅ Example configuration file

## Installation

```bash
# Configure your site
cp config.toml.example config.toml
# Edit config.toml with your database and site settings

# Run the server
chmod +x manga-blog
./manga-blog
```

Check the folders in this repository for all available versions.
